# nsMail

**nsMail** is a secure, minimal, and modern contact form solution for Next.js projects that leverages **Discord Webhooks** instead of a traditional database. It features a beautiful UI with glassmorphism design, full TypeScript support, and built-in validation.

---

## Features

- No database required — messages are delivered directly via Discord.
- Sleek and modern interface with responsive design.
- Glassmorphic styling with dark/light mode support.
- Input validation using `zod`.
- Built with Next.js App Router and Shadcn UI.

---

## Technologies Used

- [Next.js](https://nextjs.org) — App Router for server-side rendering.
- [TypeScript](https://www.typescriptlang.org) — Strong typing and DX.
- [Shadcn UI](https://ui.shadcn.com) — Accessible and elegant component library.
- [Zod](https://zod.dev) — Schema-based validation.
- [Lucide React](https://lucide.dev) — Icon system.
- [Sonner](https://sonner-ui.vercel.app) — Toast notifications.
- [Framer Motion](https://www.framer.com/motion) — Smooth animations.
